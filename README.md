# Truncate Text App ✂️📝

A simple and efficient web application that allows you to truncate text to a specified length. Perfect for shortening long strings, creating previews, or managing text content with ease! 🚀

## Features ✨
- **Input Text Area**: Paste or type your text here 📝
- **Output Text Area**: View the truncated result instantly 🔄
- **Custom Length**: Set your desired truncation length with a number input 🔢
- **Truncate Button**: Click to process the text ✂️
- **Copy Button**: Easily copy the output to clipboard 📋

## Live Demo 🌐
Check out the live version here: [https://iam269.github.io/Truncate-Text-App/](https://iam269.github.io/Truncate-Text-App/)

## How to Use 📖
1. Enter your text in the "Input Text" area
2. Set the desired length using the number input (default: 10)
3. Click "Truncate" to shorten the text
4. Use "Copy" to copy the result to your clipboard

Enjoy truncating! 😊
